# Gemini CLI - QuietChatter Project Entry

이 프로젝트에서 Gemini CLI를 사용할 때의 필수 진입 지침입니다.

## 1. 초기화 지침 (Mandatory)
- 모든 세션 시작 시, 프로젝트 루트의 GEMINI.md를 가장 먼저 읽고 그 지시에 따르십시오.
- GEMINI.md는 이 프로젝트의 모든 아키텍처 원칙, 행동 규정 및 정보 습득 순서를 정의하는 단일 출처(SSOT)입니다.
- 모든 작업 시 superpowers 스킬들을 확인하고 필요시 활성화하여 사용하십시오.

## 2. 참조 순서 준수
- GEMINI.md에 명시된 순서(GEMINI.md -> docs/requirements/prd.md -> docs/ARCHITECTURE.md 등)대로 문맥을 파악한 후 작업을 시작하십시오.
- 모든 코딩 스타일과 소통 방식(강조 서식 및 이모티콘 사용 금지 등)은 GEMINI.md의 규정을 엄격히 준수하십시오.
